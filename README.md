# Loja de Açaí

Welcome to the Loja de Açaí documentation! This README file outlines the structure of the website, brand guidelines, page descriptions, features, and setup instructions.

## Website Structure
- **Home**: Introduction to Loja de Açaí, showcasing our offerings.
- **Build-Your-Bowl**: Interactive feature for customers to create their custom bowls.
- **Bio-blueprint Subscription**: Monthly subscription service tailored to customer preferences.
- **Health Investment Dashboard**: User dashboard to track health metrics and investment in wellness.
- **About Us**: Information about our mission and values.
- **Contact**: How to reach us for inquiries.

## Brand Guidelines
- **Colors**: 
  - Primary Color: #005B5B (Dark Teal)
  - Secondary Color: #F4A261 (Light Orange)
  - Accent Color: #2A9D8F (Teal)

- **Typography**:
  - Headings: "Montserrat", Bold
  - Body: "Open Sans", Regular

- **Vibe**: Friendly, fresh, and vibrant, encouraging a community-focused approach to healthy eating.

## Page Descriptions
1. **Home**: Welcomes users and highlights the unique offerings.
2. **Build-Your-Bowl**: Promotes user interaction by allowing customization of bowls.
3. **Bio-blueprint Subscription**: Details the benefits of subscribing and how it works.
4. **Health Investment Dashboard**: A place for tracking health goals and progress.
5. **About Us**: Builds trust through transparency about our mission and our team.
6. **Contact**: Offers multiple channels for customer communication.

## Features
- **Build-Your-Bowl**: Users can select base, toppings, and extras to build their custom acai bowl.
- **Bio-blueprint Subscription**: Personalized nutrition plans delivered monthly to subscribers' doors.
- **Health Investment Dashboard**: A personalized dashboard where users can monitor their health goals and metrics over time.

## Setup Instructions
1. Clone the repository: `git clone https://github.com/danlinhnguyen2806-ctrl/lojadeacai`
2. Navigate to the project directory: `cd lojadeacai`
3. Install dependencies: `npm install`
4. Run the application: `npm start`
5. Visit [http://localhost:3000](http://localhost:3000) in your browser to view the application.
